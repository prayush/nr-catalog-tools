# Tutorials

Step-by-step, runnable walkthroughs of the main workflows:

| Tutorial | What you'll learn |
|----------|-------------------|
| [Loading a waveform](load-waveform.md) | Load a BBH merger from the RIT catalog, inspect metadata, extract physically-scaled modes, plot polarizations |
| [Cross-catalog comparison](cross-catalog.md) | Find the "same" system in RIT and SXS, align in time and phase, compute a noise-weighted mismatch |
| [NR vs surrogate comparison](surrogate-comparison.md) | Validate NRSur7dq4 against a catalog simulation with per-mode noise-weighted matches |

All tutorials assume the package is [installed](../installation.md) and download
waveform data on first run (network access required).
